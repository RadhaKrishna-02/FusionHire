# AI Interviewer System

A multimodal AI-based interview system that evaluates candidates using speech input.

## Features
- Resume upload
- Difficulty selection
- Voice-based answering
- NLP-based evaluation
- Final scoring system

## Tech Stack
- FastAPI (Backend)
- Whisper (Speech-to-text)
- Sentence Transformers (NLP)
- HTML, CSS, JS (Frontend)

## Deployment
- Backend → Render
- Frontend → Netlify"# FusionHire"
